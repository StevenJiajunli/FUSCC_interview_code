# FUSCC Interview Code

This repository contains the code and supplementary functions generated while preparing for the FUSCC interview.

The scripts were mainly used to reproduce and visualize **Fig. 1B–1E** from the study:

**“CCL19+ dendritic cells potentiate clinical benefit of anti-PD-(L)1 immunotherapy in triple-negative breast cancer.”**

## Repository Structure

- `main_function.R`  
  Main analysis and visualization functions used for reproducing Fig. 1B–1E.

- `additional_functions_for_FUSCC.R`  
  Supplementary functions and customized plotting utilities used during the analysis.

## Notes

The code was prepared for academic discussion and interview review.  
Some paths, object names, or intermediate files may need to be adjusted according to the local analysis environment.
